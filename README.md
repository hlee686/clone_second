# Cloning a spotify website using HTML, CSS, and Javascript

## 사용한 기술들
1. HTML5
2. CSS3
3. Javascript

## 구현한 페이지들
1. 메인 페이지
2. 회원가입 페이지
3. 로그인 페이지

## 작업기간
2022.12.21 ~ 2022.12.28

## 사용법
1. 메인 페이지가 보입니다.
2. 우측 상단에 Sign Up을 클릭하시면, 회원가입 페이지로 redirect합니다.
3. 우측 상단에 Log in을 클릭하시면, 로그인 페이지로 redirect합니다.
